# История изменений

Все значительные изменения в этом проекте будут задокументированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и проект придерживается [Семантического Версионирования](https://semver.org/lang/ru/).

## [2.0.0] - 2024-01-15

### Добавлено
- Полный рефакторинг кода
- Новый интерфейс с ASCII-артом краба
- Многопоточный поиск поддоменов
- Улучшенное сканирование портов
- Анализ SSL сертификатов
- Определение технологий на сайте
- Сохранение результатов в JSON
- Русскоязычный интерфейс
- Конфигурационный файл

### Изменено
- Улучшена производительность
- Оптимизирован код
- Добавлены проверки ошибок
- Улучшено форматирование вывода

### Исправлено
- Исправлены баги с WHOIS
- Исправлена обработка DNS запросов
- Исправлена многопоточность

## [1.0.0] - 2024-01-01

### Добавлено
- Первоначальный выпуск
- Базовые функции OSINT
- Простой интерфейс
- Основной функционал анализа
